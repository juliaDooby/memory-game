### Development: 

* The memory game, or concentration, as it is sometimes called, is a popular card game played by children and adults around the world, it can be seen as a game of strategy.
* In contrast to the simplicity of the optimal strategy, the analysis leading to its optimality proof is rather involved. It supplies an interesting example of concrete mathematics of the sort used in the analysis of algorithms.
